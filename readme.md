## てすさく
テストケースを作成するツールです。

## 今できること
- ユーザー登録とログイン
- テストスイートの作成
- テストケースの作成/更新/削除
- 権限(管理者・実行者)による処理分け

## 実装予定
- 不具合報告書の作成機能
- 不具合報告書へのエビデンスの添付機能
- テストケースのセル結合
- ユニットテスト
- CI/CDの環境構築
